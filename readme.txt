Greetings
1- clone the repository to your location machine.
2- create a virtual environment and activate it.
3- run the command pip install -r requirements.txt
4- run the commands (set FLASK_APP=flaskr, set FLASK_ENV=development, flask run) on windows OR (export FLASK_APP=flaskr, export FLASK_ENV=development, flask run) on Linux and Mac
5- to access the product page type on your browser "http://127.0.0.1:5000/product/".
6- you can navigate through the pages using the Side menu options on the left of the screen.

Note: the test cases are not complete, and they do have some problems.
